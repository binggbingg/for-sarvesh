UPLOAD TO GITHUB
1. Unzip this folder.
2. Upload BOTH files to the ROOT of your GitHub repository:
   - index.html
   - sarvesh-art.png
3. Commit changes.
4. Go to Settings > Pages.
5. Source: Deploy from a branch.
6. Branch: main and folder: /root.
7. Save and wait a few minutes.
